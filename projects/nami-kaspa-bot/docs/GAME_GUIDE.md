# 🌲 娜米的英雄奇幻冒險

## 遊戲簡介

這是一款基於 **Kaspa 區塊鏈** 的卡牌遊戲！

- 🎴 **召喚英雄** — 命運由區塊 hash 決定
- ⚔️ **PVP 對戰** — 擊殺敵人，搶奪獎勵
- 💰 **獎勵發放** — 存活英雄按積分分錢
- ☠️ **永久死亡** — 死了就是死了，寫在鏈上

---

## 🚀 快速開始

### 1️⃣ 創建錢包
```
/nami_wallet <PIN>
```
設定 4-6 位數字 PIN，創建你的 Kaspa 錢包

### 2️⃣ 領取 tKAS
```
/nami_faucet
```
每天可領取測試幣（mana）

### 3️⃣ 召喚英雄
```
/nami_hero <PIN>
```
消耗 10 mana，召喚一位英雄！
命運由區塊鏈決定，完全公平！

### 4️⃣ 查看英雄
```
/nami_heroes
```
查看你的英雄列表

## 💰 獎勵機制

### 獎勵池
- 每次召喚/銷毀付 10 mana 給大地之樹
- 70% 進入獎勵池
- 30% 運營費

### 發放時機
當 DAA（區塊高度）結尾為 **66666** 時發放
例如：380066666, 380166666...
（約每 2-3 小時一次）

### 分配方式
按 **積分** 比例分配給 **存活英雄**：
```
積分 = 存活天數 + 稀有度加成 + 擊殺數×2
```

稀有度加成：
- 普通 +1 / 優秀 +2 / 稀有 +3
- 史詩 +5 / 傳說 +8 / 神話 +13

## 🔍 驗證系統

每張英雄卡都有 **鏈上銘文**，可以驗證真偽：

```
/nami_verify <英雄ID>
```

驗證項目：
- ✓ 來源區塊 hash
- ✓ 屬性計算正確
- ✓ 付款給大地之樹

通過 = 🎉 **正卡**
